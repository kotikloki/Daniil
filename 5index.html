Циклы (for и while)

python # Цикл FOR (повторить 3 раза) for i in range(3): print(f"Итерация №{i}") # Цикл WHILE (выполняется, пока верно условие) count = 5 while count > 0: print(count) count -= 1 # Уменьшаем count на 1

📦 Структуры данных

Списки (List) — изменяемый набор элементов: fruits = ["яблоко", "банан", "апельсин"] Словари (Dict) — пары "ключ: значение": user = {"name": "Иван", "age": 30} Кортежи (Tuple) — неизменяемые списки: coordinates = (55.75, 37.61) Множества (Set) — только уникальные значения: unique_numbers = {1, 2, 3, 3}
